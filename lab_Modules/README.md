# Refelction

## 1. What is the purpose of type="Module" in HTML?
It allows Javascript files to use import and export statements.

## 2. Why is it useful to separate fetchUsers() into api.js?
It keeps data fetching separate and makes the code easier to manage.

## 3. Why is it useful to separate rendering into ui.js?
It keeps UI code organized and separate from application logic.

## 4.What would happen if everything stayed in one file as the project grows?
The code would become messy and harder to maintain or debug

## 5. What is the difference between exporting and importing?
Expporting makes functions available to other files, while importing allows those functions to be used in another file.
